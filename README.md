# Parking Lot Management System

This project is a simple implementation of a Parking Lot Management System in Java.

## Features

- Allows parking and leaving vehicles.
- Keeps track of available parking slots.
- Displays the current status of the parking lot.

## Usage

1. Clone the repository:

```

git clone https://github.com/your-username/parking-lot.git

```

2. Compile the Java files:

```

javac Main.java

```

3. Run the Main class:

```

java Main

```

4. Follow the on-screen instructions to park vehicles, leave slots, and view the parking lot status.
